# Welcome to @raketa-builder/ui

This storybook contains documentation and examples for all the basic building blocks of the Raketa UI library. It is used in the @raketa-builder/builder, admin interface for Next.js projects and custom input packages.

## Installation

```bash
yarn add @raketa-builder/ui
```

## Usage

```jsx
import { Button } from `@raketa-builder/ui`

<Button variant="primary">Primary action</Button>
```
